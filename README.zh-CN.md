# MCSManager Lite

[🇺🇸 English](./README.md) | 🇨🇳 简体中文 | [🇹🇼 繁體中文](./README.zh-TW.md)

一个基于 `uni-app` 与 `Vue 3` 构建的移动端优先 MCSManager 客户端。

这个项目的初衷很直接：虽然 MCSManager 已经有成熟的 Web 面板，但目前还没有一个真正围绕官方 API、适合手机日常使用的成熟移动端项目。所以这个仓库的目标，就是做一个轻量、顺手、适合触控操作的 MCSManager 移动端客户端。

## 为什么要做这个项目

`MCSManager Lite` 主要想解决这些场景：

- 在手机上快速连接 MCSManager 面板
- 按节点查看实例，而不是硬看桌面端缩小版界面
- 在移动端直接查看实例详情
- 随时看日志、发命令、管理文件
- 用更适合手机的交互方式完成日常运维操作

## 当前状态

项目目前已经具备基础可用性，但仍然在持续迭代中。

目前已经实现：

- 默认先进入登录页，并支持自动判断协议与静默补全默认端口
- 控制台总览页，可查看面板、系统和节点状态
- 按节点切换的实例列表，支持搜索、分页和快捷操作
- 实例详情页，支持运行信息、控制台、发送命令和文件管理
- 文件管理中的文本预览与图片预览
- 全局液态玻璃风格 UI
- 自定义弹层、菜单与消息反馈，而不是依赖系统级交互
- 更偏移动端体验的页面切换与标签切换动画

## 页面与功能

### 登录页

- 输入面板地址与 API Key
- 自动补全协议和默认端口
- 本地保存连接配置，方便下次直接使用

### 控制台

- 面板版本信息
- 当前节点摘要
- 在线节点数量
- 运行中的实例统计
- 基础系统信息

### 实例列表

- 节点切换
- 按名称搜索实例
- 长列表分页
- 快速执行开启、关闭、重启、终止
- 带动效进入实例详情

### 实例详情

- 内存、CPU、玩家数量等运行摘要
- 控制台日志查看
- 发送命令
- 基础实例信息
- 自定义文件管理弹层
- 文本文件预览与编辑
- 图片文件预览

### 设置页

- 查看当前连接地址和脱敏 API Key
- 查看节点摘要
- 退出登录并返回登录页

## 技术栈

- `uni-app`
- `Vue 3`
- 原生 JavaScript
- 轻量级自定义响应式状态管理

## 项目结构

```text
src/
  api/               概览、实例、文件等接口封装
  components/        共享组件，例如 tab 容器与消息宿主
  pages/             登录、控制台、实例、设置、实例详情
  stores/            轻量状态管理
  utils/             格式化、存储、URL、导航、消息等工具
static/              图标与 tabbar 资源
```

## 开发说明

当前仓库是标准 `uni-app` 项目结构，比较适合直接使用 `HBuilderX` 进行开发和预览。

常见本地流程：

1. 用 `HBuilderX` 打开项目
2. 运行到 `Web` 或移动端环境
3. 使用有效的 MCSManager 面板地址与 API Key 进行连接

## 设计方向

这个项目当前的设计重点包括：

- 移动端优先
- 面板操作路径尽量简化
- 更多自定义弹层与转场
- 以液态玻璃风格为核心的视觉语言

## 后续规划

后续可以继续补强的方向包括：

- 覆盖更多 MCSManager API 能力
- 更完整的文件管理操作
- 更丰富的运行监控视图
- 更好的状态刷新与实时性体验
- 面向移动端发布的打包与分发流程

## 声明

这是一个独立的社区项目，并非 MCSManager 官方客户端。

在生产环境中使用前，请自行确认你的部署方式、权限配置与 API 暴露策略是否安全。
