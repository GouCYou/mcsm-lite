# MCSManager Lite

[🇺🇸 English](./README.md) | [🇨🇳 简体中文](./README.zh-CN.md) | 🇹🇼 繁體中文

一個基於 `uni-app` 與 `Vue 3` 建構的行動端優先 MCSManager 用戶端。

這個專案的初衷很單純：雖然 MCSManager 已經有成熟的 Web 面板，但目前仍然缺少一個真正圍繞官方 API、適合手機日常使用的成熟行動端專案。因此這個倉庫想做的，就是一個輕量、順手、適合觸控操作的 MCSManager 行動端體驗。

## 為什麼要做這個專案

`MCSManager Lite` 主要希望解決這些情境：

- 在手機上快速連線到 MCSManager 面板
- 依節點查看實例，而不是硬看縮小版桌面後台
- 在行動端直接查看實例詳情
- 隨時查看日誌、傳送命令、管理檔案
- 用更符合手機使用方式的互動完成日常維運

## 目前狀態

目前專案已具備基礎可用性，但仍在持續迭代中。

目前已經實作：

- 預設先進入登入頁，並支援自動判斷協議與靜默補全預設連接埠
- 控制台總覽頁，可查看面板、系統與節點狀態
- 依節點切換的實例列表，支援搜尋、分頁與快捷操作
- 實例詳情頁，支援執行資訊、控制台、傳送命令與檔案管理
- 檔案管理中的文字預覽與圖片預覽
- 全域液態玻璃風格 UI
- 自訂彈窗、選單與訊息提示，而非依賴系統級互動
- 更偏向行動端的頁面切換與分頁切換動畫

## 頁面與功能

### 登入頁

- 輸入面板位址與 API Key
- 自動補全協議與預設連接埠
- 本地保存連線設定，方便下次直接使用

### 控制台

- 面板版本資訊
- 目前節點摘要
- 在線節點數量
- 執行中的實例統計
- 基礎系統資訊

### 實例列表

- 節點切換
- 依名稱搜尋實例
- 長列表分頁
- 快速執行啟動、關閉、重啟、終止
- 帶動畫進入實例詳情

### 實例詳情

- 記憶體、CPU、玩家數等執行摘要
- 控制台日誌查看
- 傳送命令
- 基礎實例資訊
- 自訂檔案管理彈窗
- 文字檔預覽與編輯
- 圖片檔預覽

### 設定頁

- 查看目前連線位址與遮罩後的 API Key
- 查看節點摘要
- 登出並返回登入頁

## 技術棧

- `uni-app`
- `Vue 3`
- 原生 JavaScript
- 輕量自訂響應式狀態管理

## 專案結構

```text
src/
  api/               概覽、實例、檔案等 API 封裝
  components/        共用元件，例如 tab 容器與訊息宿主
  pages/             登入、控制台、實例、設定、實例詳情
  stores/            輕量狀態管理
  utils/             格式化、儲存、URL、導覽、訊息工具
static/              圖示與 tabbar 資源
```

## 開發說明

目前倉庫是標準 `uni-app` 專案結構，特別適合直接用 `HBuilderX` 開發與預覽。

常見本地流程：

1. 使用 `HBuilderX` 開啟專案
2. 執行到 `Web` 或行動端環境
3. 使用有效的 MCSManager 面板位址與 API Key 進行連線

## 設計方向

這個專案目前的設計重點包括：

- 行動端優先
- 面板操作流程盡量簡化
- 更多自訂彈窗與轉場
- 以液態玻璃風格為核心的視覺語言

## 後續規劃

後續可以持續補強的方向包括：

- 覆蓋更多 MCSManager API 能力
- 更完整的檔案管理操作
- 更豐富的執行監控視圖
- 更好的狀態刷新與即時體驗
- 面向行動端發佈的打包與分發流程

## 聲明

這是一個獨立的社群專案，並不是 MCSManager 官方用戶端。

在正式環境使用前，請自行確認你的部署方式、權限設定與 API 暴露策略是否安全。
